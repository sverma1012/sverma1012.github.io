Writing Programs on GitHub

GitHub is a graphical interface for Git, which is a command line tool. It allows users to work on projects individually and collaboratively from anywhere with various features.

Over the summer of 2019, I researched small projects to engage in to improve and hone my understanding of programming and Python. Through this project, I hope to advance my programming skills by laying a strong foundation.

Having taken multiple computer science course, I understand the importance and value of programming in the technical industry. While I learned Python through my college classes, I wished to lay a stronger base to my knowledge and, thus, decided to start a project to write small programs in Python. Further, since GitHub is a valued and flexible platform where people can share and update their code, I decied to publish my code on on GitHub as an attempt to understand the platform better.
